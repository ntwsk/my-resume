# Nuttawut Sukaew — Resume Site

Personal resume site for **Nuttawut Sukaew** ([@ntwsk](https://github.com/ntwsk)) — Salesforce Developer @ ATA IT Limited (National Bank of Canada group).

🌐 **Live:** https://ntwsk.github.io/my-resume/

> 100% vibe-coded with Claude Code — zero lines written.

---

## Quick start

```bash
npm install
npm run dev       # → http://localhost:5173/my-resume/
npm run build     # production bundle → dist/
npm run deploy    # build + push to gh-pages branch
```

## Stack

React 19 · Vite 8 · Tailwind CSS v4 · Framer Motion · GSAP · Lenis · GitHub Pages

## Where to edit content

All resume content lives in **`src/data/resume.js`** — profile, skills, experience, certifications, education, stages.
